# InstaPlugins
A new Roblox Framework.

# What is InstaPlugins
InstaPlugins is a Luau Plugin that makes Scalable and Efficient the creation of Plugins, it is inspired from ReactJs. It is easy to import, config and init.

# How to use it
When you run InstaPlugins in Roblox Studio you need to choose a Name for the Plugin and choose a place where to import the Framework (i suggest ServerScriptService), you can also toggle turn on/off the Shortcuts, which lets you can write !setup or similiar keywords in
